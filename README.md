Welcome to the Knowledge Gate! This project is a sleek, modern, and interactive user interface for a robot learning portal. The goal of this UI is to provide a gateway where a robot can eventually ask for book recommendations about Earth.

This project focuses on HTML5, CSS3 Flexbox, the Box Model, and Advanced CSS Animations.
